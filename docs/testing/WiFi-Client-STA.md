Tests for Wi-Fi client (station) mode, where Flipper One connects to an external access point — the role laptops and smartphones normally play.

## Tests

1. Connect to a Wi-Fi access point as a client.
2. Measure throughput with `iperf3`.
3. Repeat across all bands (2.4 GHz, 5 GHz, 6 GHz).

Simultaneous client + access point operation is covered on the [AP+STA SCC](./WiFi-AP-STA-SCC.md) page.
