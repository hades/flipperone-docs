Tests for wired Ethernet connectivity on Flipper One, provided through USB and other network adapters.

## Tests

1. Connect a USB Ethernet adapter and confirm the interface comes up.
2. Obtain an IP address over DHCP and confirm connectivity.
3. Measure throughput with `iperf3`.
4. Test the link speeds supported by the adapter (10 / 100 / 1000 Mbps).
