This section covers testing and validation of all network interfaces on Flipper One: wireless, wired, cellular, and satellite.

## Test categories

- **[Wi-Fi](./WiFi.md)** — built-in Wi-Fi 6E module. Access point, client, concurrency, and dual-band test modes.
- **[Ethernet](./Ethernet.md)** — wired networking over USB and other adapters.
- **[Cellular modems](./Cellular-Modems.md)** — M.2 cellular modems (LTE / 5G).
- **[NTN satellite modems](./Satellite.md)** — non-terrestrial network (satellite) connectivity.
- **[Bluetooth](./Bluetooth.md)** — Bluetooth and BLE, integrated in the same MT7921 combo chip as Wi-Fi.
