# Markup example

На этой странице содержатся примеры разметки, которые можно использовать в этой wiki. Оно поддерживает базовый Markdown, но есть так же специфический синтаксис Archbee.

This wiki is powered by Archbee documentation system: https://archbee.com connected to public Github repo https://github.com/flipperdevices/flipper-one-docs. It means everyone can suggests edits of any any page, and if Flipper team will approve this edits, it will appears в этой документации. 

## Github sources

Исходные тексты этой wiki хранятся в гитхаб репозитории https://github.com/flipperdevices/flipper-one-docs Чтобы отредактировать любую страницу в этой wiki, создайте pull request с вашми изменениями: LINK How to make pull request to this wiki

После каждого изменения файлов в гитхаб репозитории, эта вики перегенириуется и изменения применяются на сайте.
