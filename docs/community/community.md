Flipper One has an active community of developers building tools, libraries, integrations, and guides around it. This page lists community-driven projects related to Flipper One development.

:::hint{type="warning"}
**Community projects are independent of Flipper Devices.**

The projects listed here are created and maintained by community members, not by Flipper Devices. Flipper Devices does not own, endorse, audit, or maintain them, and makes no guarantees about their quality, safety, or continued availability. Use them at your own discretion.
:::

***

## Contributing a project

If you've built something around Flipper One and would like it listed here, open a pull request to the [flipperone-docs](https://github.com/flipperdevices/flipperone-docs) repository and add your project to this page.

To keep the list useful, include:

- **Project name** — a short, descriptive name.
- **Link** — a URL to the repository, release page, or documentation.
- **Description** — one or two sentences on what it does and who it's for.
- **Author** — your GitHub username or handle.

Follow the [contribution guide](../resources/docs/About-Docs.md#how-to-contribute) for pull request instructions.

## Projects

### Flipper One UI Lab

A container-based lab for running and inspecting the public Flipper One Linux userspace and `fake-flipctl2` UI harness from a desktop browser — no physical hardware required. Provides a device-shell overlay, an API hooks panel for mocking device responses, and two images: a fast dev image and a rootfs-based image closer to real device userspace.

- Author: kalicyh
- GitHub: https://github.com/kalicyh/flipperone-lab
